# Termux-Setting

#### * Hi, if you want to set up a termux quickly, here you are!
---
## Installing for termux.

* `pkg install git`
* `git clone https://github.com/mishakorzik/Termux-Setting`
* `cd Termux-Setting`
* `bash Term-sett.sh`

#### Everything is ready!  Now we are waiting for everything to download!

* `Succes!`
